Calendário Simples

Este complemento oferece um calendário acessível, rápido e cheio de recursos para usuários do NVDA. Além de navegar pelas datas, você pode gerenciar notas, consultar fases da lua, verificar feriados e calcular dias restantes para o fim do ano.

Como Abrir

Para abrir o calendário a partir de qualquer lugar no Windows:

  • Pressione: NVDA + Shift + C

Você também pode encontrá-lo no menu Ferramentas do NVDA.

Teclas de Navegação

Use estas teclas para se mover pelo calendário:

Tecla Ação
Seta Esquerda / Direita Dia anterior / Dia seguinte.
Seta Cima / Baixo Semana anterior / Próxima semana.
Page Up / Page Down Mês anterior / Próximo mês.
Home Vai para o primeiro dia do mês atual.
End Vai para o último dia do mês atual.
Ctrl + Home Recua um ano.
Ctrl + End Avança um ano.

Comandos e Ferramentas

Com o calendário aberto, utilize as teclas abaixo para acessar funções rápidas (não é necessário pressionar Ctrl):

Tecla Função
H Hoje: Vai imediatamente para a data atual.
G Ir para Data: Abre uma caixa para digitar uma data específica (ex: 25/12/2025).
F Fases da Lua: Anuncia a fase da lua atual e o período (data de início e fim da fase).
D Dias Restantes: Informa quantos dias faltam para terminar o ano.
L Lista de Feriados: Exibe uma lista com todos os feriados fixos e móveis do ano selecionado.
C Copiar: Copia a data selecionada para a área de transferência.
Enter Notas: Abre um campo para adicionar ou editar uma nota pessoal para o dia selecionado.
F1 Exibe a ajuda rápida dentro do programa.
Esc Fecha o calendário.
Dica: O calendário emite sons diferentes para ajudar na orientação. Um bipe agudo indica que você passou pelo dia de hoje.
